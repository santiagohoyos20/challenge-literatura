# Literatura

Aplicación de consola que permite a los usuarios interactuar mediante texto para buscar, consultar y gestionar información de libros obtenida desde una API pública, desarrollada en el marco del programa de formación Oracle Next Education junto con Alura Latam.
desarrollado en el marco del programa de formación Oracle Next Education junto con Alura Latam.

La aplicación consume datos desde la API de Gutendex, la cual proporciona acceso a libros del proyecto Project Gutenberg.

El sistema permite realizar búsquedas dinámicas, almacenar información en base de datos y consultar registros previamente guardados.

### Objetivo

Desarrollar una aplicación de consola que:

- Consuma datos desde una API externa.
- Permita interacción textual con el usuario.
- Ofrezca al menos 5 opciones de interacción.
- Gestione persistencia de datos.
- Aplique buenas prácticas de desarrollo backend.

### Tecnologías Utilizadas

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- API REST (Gutendex)
- Maven

### Funcionalidades Principales

El sistema presenta un menú interactivo en consola con las siguientes opciones:

1. Buscar libro por título (consulta a la API).
2. Listar libros registrados en la base de datos.
3. Listar autores registrados.
4. Listar libros por idioma.
5. Listar autores vivos en un determinado año.
